changeInsertIssueModeAPI

changeInsertIssueModeAPI is an API that activates the issue function. This API is a parameter flag is Boolean type, if true, issue mode is activated and if false, it is deactivated. Active If you click the object you want to register the issue in, the information of the object is displayed, and the issue registration callback function can be customized by the user.

Parameter
name type description
managerFactoryInstance ManagerFactory start of mago3D
flag Boolean true: active, false: Inactive
