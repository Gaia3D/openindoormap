changeColorAPI

This is an API that changes the color of the desired data or data object on a project basis.

Parameter
name type description
managerFactoryInstance ManagerFactory start of mago3D
projectId String Project ID
dataKey String Data unique key
objectIds String[] object id. If multiple, enter with ','
property String Property value ex) isMain=true
color String Receives a string value that connects R, G, and B colors with ','.
