gaia3d.JS V1.0.5



Open source JavaScript library for 3D multi-block visualization

Next-generation 3D GIS platform that integrates and visualizes AEC (Architecture Engineering Construction) domain and traditional 3D spatial information (3D GIS). Integrate AEC and 3D GIS into web browsers that cannot be distinguished indoors and outdoors.

Browse and collaborate on large Building Information Modeling (BIM), Jupiter Tessellation (JT) and 3D GIS files without installing a program in a web browser.

Development environment

Java8, eclipse, neon, node, apache, jasmine, jsdoc, Gulp, eslint, Jquery

Getting Started


  1. Source Installation
    • github It accepts mago3DJS from.
    • Installation path C://git/repository/mago3djs
  2. Node Installation
    • npm install
    • npm install -g gulp
  3. Node Server Execution
    • Running a local server: C:\git\repository\mago3djs > node server.js
    • Running a public server: C:\git\repository\mago3djs > node server.js --public true
  4. Create Symbolic Link
    • Create Data Folder: mklink /d "C:\git\repository\mago3djs\data" "C:\data"
    • Delete data folder: C:\git\repository\mago3djs > rmdir data
