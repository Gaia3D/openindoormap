changeLightingAPI

API to adjust the brightness of the selected object.

Parameter
name type description
managerFactoryInstance ManagerFactory start of mago3D
ambientReflectionCoef Number Reflectance range, not direct light. Default 0.5
diffuseReflectionCoef Number Magnetic color reflectance range. Default 1.0
specularReflectionCoef Number Surface gloss range. Default 1.0
ambientColor String Reflectance RGB not direct light, Connect with commas
specularColor String The gloss color of the surface. RGB, comma connection
