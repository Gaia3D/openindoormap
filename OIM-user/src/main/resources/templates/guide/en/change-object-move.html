changeObjectMoveAPI

API to change the moving mode of an object. Building when the move mode (objectMoveMode) value is 0 (All) The whole is selected and moved, and if 1 (Object), it can be selected and moved in units of objects, such as a window, and if it is 2 (None) The object is not selected and the screen is moved.

Parameter
name type description
managerFactoryInstance ManagerFactory start of mago3D
objectMoveMode Number 0 = All, 1 = Object, 2 = None
